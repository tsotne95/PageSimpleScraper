# PageSimpleScraper

Web Page simple scraper using **Jsoup**  

and for url validate i'm using **Apache Commons UrlValidator class**

for unique images and links it create tab delimited file with details on it:

images:
- image number
- image url
- image width
- image height
- image alt  

links:
- link url
- link text
